# Auditoría de seguridad — Empresa ficticia

Caso práctico académico realizado como parte del Certificado Profesional de Ciberseguridad de Google.

## Objetivo
Evaluar controles de seguridad, identificar riesgos relevantes y documentar requisitos de cumplimiento aplicables a una empresa ficticia.

## Contenido
- Alcance y objetivos de auditoría
- Evaluación de controles
- Evaluación de riesgos
- Lista de control de cumplimiento normativo

## Competencias demostradas
- Identificación y evaluación de riesgos
- Controles administrativos, técnicos y físicos
- Cumplimiento normativo
- Documentación de auditoría
- Redacción de recomendaciones de seguridad

> Todos los datos, sistemas y organizaciones incluidos son ficticios y se utilizan únicamente con fines educativos.
