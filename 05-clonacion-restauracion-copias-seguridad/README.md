# Práctica 05 - Clonación de discos, gestión y formateo de particiones en entornos Linux y planificación / gestión de copias de seguridad
Documentación técnica de una práctica de laboratorio relacionada con la clonación de discos con Clonezilla, creación, formateo, montaje y eliminación de particiones en Linux mediante la línea de comandos y herramientas gráficas como GParted.
Creación, manipulación y restauración de archivos de copia de seguridad. Sistemas RAID.

El PDF de la práctica se encuentra en esta carpeta.
