Portfolio ASIR y Ciberseguridad

Este repositorio reúne una selección de prácticas técnicas realizadas durante mi formación en Administración de Sistemas Informáticos en Red, con orientación hacia sistemas, redes y ciberseguridad.

El objetivo de este portfolio es mostrar evidencias prácticas de trabajo en entornos de laboratorio, incluyendo administración de sistemas, virtualización, redes, Active Directory, Linux, Windows, análisis de logs y documentación técnica.

Durante mi formación he trabajado con laboratorios prácticos relacionados con sistemas operativos, hardware, redes, máquinas virtuales y análisis básico de seguridad.

Áreas trabajadas: 

- Administración de sistemas Linux.
- Administración de sistemas Windows.
- Configuración de máquinas virtuales.
- Instalación y configuración de sistemas operativos.
- Active Directory y gestión de usuarios.
- Fundamentos de redes.
- Simulación de redes con Cisco Packet Tracer.
- Revisión de logs y auditoría.
- Documentación técnica con capturas y evidencias.
